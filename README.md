# 🚢 Titanic Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on the Titanic dataset to uncover the factors that influenced passenger survival. The analysis includes data cleaning, statistical summaries, visualizations, and insights obtained through univariate and bivariate analysis.

The goal of this project is to understand the dataset, identify meaningful patterns, and develop data analysis skills using Python.

---

## 📂 Dataset

The dataset contains information about **891 Titanic passengers**, including:

* Passenger ID
* Survival Status
* Passenger Class
* Name
* Sex
* Age
* Siblings/Spouses Aboard
* Parents/Children Aboard
* Ticket Number
* Fare
* Cabin
* Embarked Port

## 📋 Project Workflow

### 1. Data Loading

* Imported the dataset using Pandas.

### 2. Data Understanding

* Explored dataset structure.
* Checked data types.
* Examined missing values.
* Generated descriptive statistics.

### 3. Data Cleaning

* Handled missing values.
* Prepared a cleaned dataset for analysis.

### 4. Exploratory Data Analysis (EDA)

The following analyses were performed:

* Overall Passenger Survival Rate
* Survival Rate by Gender
* Survival Rate by Passenger Class
* Survival Rate by Age Group
* Survival Rate by Age
* Survival Rate by Embarkation Port
* Survival Rate by Fare
* Survival Rate by Family Size

Each analysis includes visualizations and corresponding insights.

## 🔍 Key Insights

Some important findings include:

* Female passengers had a significantly higher survival rate than male passengers.
* First-class passengers were more likely to survive than passengers in lower classes.
* Passenger age showed noticeable differences in survival patterns.
* Fare and passenger class were strongly related to survival.
* Family size influenced the likelihood of survival.
* The embarkation port also showed variations in passenger survival.

---

## 🎯 Conclusion

This project demonstrates how Exploratory Data Analysis (EDA) helps identify meaningful patterns and relationships within a dataset before applying machine learning models.

The analysis also highlights the importance of data cleaning, visualization, and statistical interpretation in the data analysis process.
